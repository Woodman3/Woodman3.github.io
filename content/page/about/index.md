# 求学和技术

本科是一个四线城市的四非大学，入学时是化学系的，后来想转去计算机失败了，曲线救国去了数学系，但是数学也是学的依托捏。毕业的时候成绩是全系倒数前几名。

技术主要点在了高性能计算和有限元分析上。

使用的语言大部分是cpp和py，自己写的东西还会使用rust。曾经用haskell写过一个lisp解释器，但是除此之外就再也没写过别的了，感觉以后rust也会步入haskell的后尘。

# 这个博客

创建这个博客的初衷是为了记录一些技术上的东西、踩过的坑和一些感想的。但是现在看来，大部分东西还是我的牢骚和唠叨。

我得承认自己不仅沟通能力不行，表达水平也好不到哪儿去，说话吐字不清写的东西也前言不搭后语。

但是我还是想写，我知道自己十几年后看这些东西大概率还是会尴尬，就像我现在看我几年写的日记和空间说说一样。但是我不希望十几年后的我忘记现在的自己，感觉那样我就白活了。

记录下自己曾经是如何的燃烧，又是如何熄灭。
